The form below can be used to enter set scores.

Incomplete sets may also be recorded and in situations where a tiebreak decider was played INSTEAD of playing a whole set, please enter the set score as 1-0.

NOTE: If this is a league or a knockout match, please don't use this form to enter the result. Instead, please go to the appropriate competition page, and then choose 'Enter Result' next to the match for which you wish to enter a score.

Please enter the region you would like to play - please only enter the first part of the postcode, e.g. SW6
Start typing and wait for suggestions. Suggestions are given in order of popularity and are limited to 25 options - please enter more letters to narrow the selections.
